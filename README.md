# UniMove Prototype

UniMove is an interactive front-end prototype for a university public-transport application. It presents three role-based experiences—Commuter, Administrator, and Transport Operator—within an iPhone 17-style interface.

This project is a visual and interaction prototype only. It does not connect to a database, process real payments, or send live transport data.

## Features

- Role-based commuter, administrator, and transport-operator portals
- Form validation for required fields, email addresses, passwords, matching password confirmation, and feed URLs
- Commuter journey planning, live-tracking views, wallet, QR ticket, rewards, notifications, and preferences
- Administrator tools for user accounts, operator data feeds, announcements, reports, and performance monitoring
- Transport-operator tools for live feed configuration and commuter service alerts
- Purple-and-blue UniMove branding with responsive iPhone 17 UI styling

## How to Open the Prototype

1. Download this repository by selecting **Code** → **Download ZIP** on GitHub.
2. Extract the downloaded ZIP file.
3. Open the extracted folder.
4. Double-click `index.html`, or open it in a modern browser such as Safari, Chrome, or Microsoft Edge.
5. Select a role and use the on-screen buttons to explore the prototype.

No installation, server, database, or internet connection is required after downloading the files.
